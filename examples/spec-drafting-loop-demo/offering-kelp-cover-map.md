# Kelp Cover Map of Three Transects

What we're bringing: a small dataset of kelp cover percentages from three
near-shore transects, sampled at 12 points each, in May 2026. Each sample
point has lat/lon, kelp species observed, and a 0-100% cover estimate.

What we'd like to see exist: a small CLI that takes the JSON of these
samples and emits a markdown receipt with one section per transect,
showing the cover estimate per sample point and a transect-level average.

The receipt should reference the data's provenance (May 2026 samples from
three named transects) and should NOT make claims about kelp population
trends — we only have one month of data.
