# Consent / Refusal / Review Desk

This sample records permission, refusal, withdrawal, and review routing without exposing protected content.

Use it when an offering is useful but cannot be bundled, displayed, summarized, or sent to AI yet.

## Sample Records

| Item | State | Review Route | Notes |
|---|---|---|---|
| Public sample commitment | public_sample | none | Can be used in Open Kit demo. |
| Participant story | review_required | contributor / facilitator | Do not display yet. |
| Protocol-bound material | protect_and_preserve | contributor-named authority | Record boundary only. |
| Withdrawn receipt | do_not_use | data steward / contributor | Tombstone only if contributor wants it. |
