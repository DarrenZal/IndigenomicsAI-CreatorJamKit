# Handoff Packet Studio

This sample turns a selected bundle into clear build attempt instructions.

The packet should answer:

- What are we trying to build or show?
- Which inputs are authorized?
- Which inputs are explicitly not authorized?
- How will we know it worked?
- What must cause a stop or refusal?
- What should the builder, reviewer, and witness produce?

Start from `templates/build-attempt-instructions.md`.
