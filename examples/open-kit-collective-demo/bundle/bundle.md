# Bundle: Open Kit Collective Demo

Status: selected for simulated build attempt

## Purpose

Show how several participant offerings can become one bounded collective build attempt:

- a public commitment
- a refusal boundary
- a commitment receipt prototype
- a clarity checklist
- consent/withdrawal constraints
- optional TELUS notes-only harness support

## Why This Bundle Was Selected

This is the safest first collective bundle. It can run with public ecological/sample material, avoids protected content, and already has an Open Kit demo path.

## What This Is Not

- not a production pool
- not a public marketplace launch
- not a monitoring program
- not ceremony
- not a claim of legitimacy, authority, certification, or reuse readiness
