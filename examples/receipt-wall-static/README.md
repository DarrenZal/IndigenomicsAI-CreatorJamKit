# Receipt Wall Static

Static sample witnessing surface for receipts, decisions, refusals, and unresolved work.

Use sample or explicitly display-approved receipts only.

## Display Rules

- No real receipt appears without display approval.
- Display is not endorsement.
- Display is not certification.
- Refused and not-run states remain visible.

## Sample Wall

| Item | Status | Display State |
|---|---|---|
| Open Kit Collective Demo | selected | sample display approved |
| Consent / Review Desk | review surface | sample display approved |
| Youth Story Map | deferred | review required |
| Real participant receipt | not displayed | display approval missing |
