# Bundle: Consent / Review Desk

Status: active candidate, not selected for build execution

## Purpose

Provide a visible place for refusal, review routing, consent state, attribution preferences, withdrawal, tombstoning, and display approval.

## Included Offerings

- protocol steward boundary note
- data steward consent/withdrawal rules
- evaluator criteria for review-required states

## Why It Remains Separate

This is a governance and preservation surface, not a build target. It should constrain every bundle without becoming an AI execution prompt.
