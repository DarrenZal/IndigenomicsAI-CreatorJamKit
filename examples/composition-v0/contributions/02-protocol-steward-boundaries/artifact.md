## Working Title

Review Required Boundary Note

## Offering

A reusable stop condition:

```text
If material is cultural, protocol-bound, private, language-related, ceremony-related, Nation-specific, or authority-bound, do not process it with AI. Record that review is required and ask the contributor who should be consulted.
```

## Possible Connection

This belongs in the Consent / Review Desk bundle. It can also appear as a refusal boundary in any build attempt instructions.
