# Creator Jam Games

Games in this directory teach Creator Jam coordination patterns through play. They use the same primitives as the kit: offerings, claims, commitments, refusals, speech-act transitions, witness receipts, review gates, and obstruction markers.

The game state is not automatically a real-world record. A game move can become a fixture or real record only through an explicit transition, authority check, and receipt.

## Current Games

| Game | Format | What It Teaches |
| --- | --- | --- |
| [Speech-Act Bingo](coordination-canoe-speech-act-bingo/) | Facilitator one-pager plus bingo card. | How to notice category shifts like dream to commitment, citation to claim, and receipt to story. |

## Boundary

Play is allowed to be playful. It is not consent, ceremony, authority, publication approval, or commitment by default.
