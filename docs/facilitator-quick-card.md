# Facilitator Quick Card

Use these questions before any technical template.

1. What are you offering?
2. What do you hope it helps with?
3. What should it not be used for?
4. Is anything private, protected, or review-required?
5. Who should be credited or asked?

Facilitator notes:

- "Does not fit yet" is a valid outcome.
- A refusal can be a complete offering.
- Protected material does not need to be disclosed for a review boundary to be recorded.
- Do not send protected or authority-bound material into AI for checking.
